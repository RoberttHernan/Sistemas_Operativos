# Sistemas operativos
Este repositorio está enfocado en el estudio teórico y práctico de los Sistemas Operativos, reuniendo apuntes, explicaciones y ejemplos de implementación de los principales conceptos que los componen.

Incluye materiales sobre:
- Gestión de procesos e hilos
- Planificación de CPU
- Sincronización y concurrencia
- Gestión de memoria
- Sistemas de archivos
- Entrada/Salida y dispositivos
- Conceptos de kernel y llamadas al sistema

Los ejemplos están desarrollados principalmente en C, Bash y otros lenguajes de bajo nivel, con un enfoque educativo que facilita la comprensión del funcionamiento interno de un sistema operativo moderno.

Este repositorio está pensado como material de apoyo para estudiantes de informática, ingeniería en sistemas y autodidactas interesados en entender cómo funcionan los sistemas operativos desde una perspectiva técnica y práctica.
